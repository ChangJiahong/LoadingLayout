# LoadingLayout
LoadingLayout TestDemo
## 可以快速、简单的切换加载中、错误、空、或者自定义界面的控件
### 使用方法：
```
步骤1.将JitPack存储库添加到构建文件中

将其添加到存储库末尾的根build.gradle中：

	allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
步骤2.添加依赖项

	dependencies {
	        implementation 'com.github.ChangJiahong:LoadingLayout:v0.1'
	}
 ```
